# Perancangan Sistem Informasi Pengisian KRS

Project ini merupakan tugas Analisis Kebutuhan Sistem yang membahas perancangan Sistem Informasi Pengisian KRS berbasis web.

## Fitur Utama
- ERD (Entity Relationship Diagram)
- Kamus Data
- HIPO (Hierarchy Input Process Output)
- User Interface Design
- Database MySQL

## Tools
- MySQL
- phpMyAdmin
- Draw.io
- Figma / AI UI Design

## Struktur Sistem
Sistem dirancang untuk:
- login mahasiswa
- pengisian KRS
- validasi mata kuliah
- persetujuan dosen PA
- monitoring status KRS
